# youtube-clone
🎬 YouTube UI Clone
A simple and responsive YouTube homepage clone built using HTML and CSS.
This project focuses on recreating the layout and design of YouTube using modern CSS techniques like Flexbox and Grid.

📌 Features
📺 YouTube-like homepage layout
📱 Responsive design (works on different screen sizes)
🎯 Clean and structured UI
🧩 Built using Flexbox and CSS Grid
🔍 Navigation bar with icons and search box

📂 Project Structure
youtube-clone/
│── index.html
│── style.css
│── youtube_images/

💡 What I Learned
Structuring web pages using semantic HTML
Creating layouts using Flexbox and Grid
Styling components to match real-world UI
Organizing assets and project files
